# Learn Node.js

This repository contains my basic Node.js practice work.

The goal of this project is to understand how Node.js works by practicing core backend concepts without using Express.js or any framework.

## What I Practiced

- Node.js HTTP server
- Request and response handling
- HTTP methods
- URL and query parameters
- Reading files using `fs`
- Writing files using `fs`
- JSON data handling
- Status codes
- Response headers
- Basic CRUD operations
  - Create
  - Read
  - Update
  - Delete

## Technologies

- Node.js
- TypeScript
- HTTP module
- File System module

```bash
npm install
